\section*{Abstract}
This document is a complete engineering post-mortem of an IMDb review downloader that failed due to IMDb architectural changes.

\section{Historical Context}
IMDb reviews were historically server-rendered HTML.

\section{Client-Side Rendering}
IMDb migrated to Next.js and GraphQL.

\section{Persisted Queries}
IMDb uses persisted GraphQL queries with SHA-256 hashes.

\section{Session Enforcement}
Requests require valid cookies.

\section{Pagination}
Cursor-based pagination using endCursor.

\section{Schema Instability}
Fields may be missing.

\section{Conclusion}
Failures were architectural, not logical.
